Chapter 1: Introduction to JavaScript

JavaScript is a versatile, high-level programming language that is primarily used to create interactive effects within web browsers. It allows you to implement dynamic features on web pages—anything from simple animations to complex web applications.

Example: Hello World


// This is a simple JavaScript program that prints "Hello, World!" to the console.
console.log("Hello, World!");

The above code uses console.log() to output text to the browser's console.

Chapter 2: Variables

Variables are containers for storing data values. In JavaScript, you can declare variables using var, let, or const.

Example: Declaring Variables


// Using let and const (ES6+)
let name = "John"; // can be reassigned
const age = 30;    // constant value, cannot be reassigned

console.log(name);
console.log(age);

Chapter 2: Data Types

JavaScript provides several data types including:

  • Number: Numeric data (e.g., 42, 3.14)
  • String: Text (e.g., "Hello")
  • Boolean: true or false
  • Object: Collections of properties
  • Undefined: A variable that has not been assigned a value
  • Null: Represents no value

Example: Data Types


let num = 42;              // Number
let str = "JavaScript";    // String
let isTrue = true;         // Boolean
let nothing = null;        // Null
let notDefined;            // Undefined

console.log(typeof num);       // "number"
console.log(typeof str);       // "string"
console.log(typeof isTrue);    // "boolean"
console.log(typeof nothing);   // "object" (a known quirk)
console.log(typeof notDefined); // "undefined"

Chapter 3: Operators

Operators are used to perform operations on variables and values. JavaScript supports arithmetic, comparison, logical, and assignment operators.

Example: Arithmetic Operators


let a = 10;
let b = 5;

console.log(a + b); // Addition: 15
console.log(a - b); // Subtraction: 5
console.log(a * b); // Multiplication: 50
console.log(a / b); // Division: 2

Chapter 4: Conditionals

Conditional statements control the flow of code based on conditions. The most common are if, else if, else, and switch.

Example: if...else


let score = 85;

if (score >= 90) {
  console.log("Grade: A");
} else if (score >= 80) {
  console.log("Grade: B");
} else {
  console.log("Grade: C or below");
}

Chapter 5: Loops

Loops allow you to execute a block of code repeatedly. JavaScript supports for, while, and do...while loops.

Example: for Loop


for (let i = 0; i < 5; i++) {
  console.log("Iteration " + i);
}

Chapter 6: Functions

Functions are blocks of code designed to perform a particular task. They are executed when "called."

Example: Function Declaration


function greet(name) {
  return "Hello, " + name + "!";
}

console.log(greet("Alice")); // Outputs: Hello, Alice!

Chapter 7: Objects

Objects are collections of key-value pairs. They allow you to store structured data.

Example: Creating an Object


let person = {
  name: "Alice",
  age: 25,
  greet: function() {
    return "Hi, I'm " + this.name;
  }
};

console.log(person.greet());

Chapter 7: Arrays

Arrays are ordered collections of values.

Example: Creating an Array


let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // Outputs: Apple

Chapter 8: DOM Manipulation

The Document Object Model (DOM) represents the web page so that programs can change the document structure, style, and content dynamically.

Example: Changing Content


// HTML Element: 
Original Content

document.getElementById("demo").innerText = "Content updated!";

In this example, the content of the element with the id demo is updated. To test this, ensure you have an element with id="demo" in your HTML.

Chapter 10: Asynchronous JavaScript

Although JavaScript is single-threaded, it can handle asynchronous operations using callbacks, Promises, and async/await.

Example: Using async/await with a Delay


function delay(ms) {
  return new Promise(resolve => setTimeout(resolve, ms));
}

async function asyncExample() {
  console.log("Start");
  await delay(1000); // Wait for 1 second
  console.log("1 second later");
}

asyncExample();

Chapter 11: Advanced Topics

This chapter covers more advanced JavaScript topics such as closures, prototypes, and ES6+ features like arrow functions and classes.

Example: Closures, Arrow Functions & Classes


// Arrow Function Example
const add = (a, b) => a + b;
console.log(add(5, 7)); // Outputs: 12

// Closure Example
function makeCounter() {
  let count = 0;
  return function() {
    count++;
    return count;
  };
}

const counter = makeCounter();
console.log(counter()); // 1
console.log(counter()); // 2

// ES6 Classes and Inheritance
class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    console.log(`${this.name} makes a noise.`);
  }
}

class Dog extends Animal {
  speak() {
    console.log(`${this.name} barks.`);
  }
}

const d = new Dog("Mitzie");
d.speak(); // Mitzie barks.
