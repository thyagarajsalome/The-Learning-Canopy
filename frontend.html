Chapter 1: HTML - Introduction

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It structures the content on the web.

Example: Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is a simple HTML page.</p>
  </body>
</html>

Explanation: This code defines a basic HTML document with a DOCTYPE declaration, head, and body sections. The <h1> tag defines a main heading and the <p> tag defines a paragraph.

Chapter 1: HTML - Elements & Structure

HTML consists of a variety of elements defined by tags. Elements can have attributes that provide additional information.

Example: Using Attributes

<a href="https://www.example.com" target="_blank">Visit Example</a>

Explanation: The <a> tag creates a hyperlink. The href attribute specifies the destination URL, and the target="_blank" attribute makes the link open in a new tab.

Chapter 1: HTML - Forms & Media

Forms allow user input. HTML forms support various input types, making it possible to capture different kinds of data.

Example: Simple Form

<form action="/submit" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>

Explanation: This form collects a user's name and submits the data using the POST method. The <label> tag improves accessibility by associating text with form controls.

Chapter 2: CSS - Introduction

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows you to apply styles to HTML elements.

Example: Basic CSS

body {
  background-color: #f0f0f0;
  color: #333;
  font-family: Arial, sans-serif;
}

Explanation: This CSS snippet sets a background color, text color, and font for the entire page.

Chapter 2: CSS - Selectors & Properties

CSS selectors target HTML elements. Properties define the styling applied to those elements.

Example: Class and ID Selectors

/* ID selector */
#header {
  background-color: #333;
  color: #fff;
}

/* Class selector */
.button {
  padding: 10px 15px;
  border: none;
  background-color: #007BFF;
  color: #fff;
}

Explanation: The ID selector targets an element with a specific ID, while the class selector targets all elements with a specified class attribute.

Chapter 2: CSS - Flexbox & Grid

Flexbox and Grid are powerful layout systems in CSS for designing responsive, modern layouts.

Example: Flexbox Layout

/* Flexbox Container */
.container {
  display: flex;
  justify-content: space-around;
  align-items: center;
}

.item {
  background-color: #007BFF;
  color: #fff;
  padding: 20px;
  margin: 10px;
}

Explanation: This code creates a flex container that centers its items and distributes space around them.

Chapter 2: CSS - Responsive Design

Responsive design ensures web pages render well on different devices. Media queries allow you to apply styles based on device characteristics.

Example: Media Query

@media (max-width: 768px) {
  body {
    background-color: #e0e0e0;
  }
}

Explanation: This media query changes the background color when the viewport width is 768px or less.

Chapter 3: JavaScript - Basics & Syntax

JavaScript is a programming language that enables interactive web pages. It can manipulate the DOM, handle events, and much more.

Example: Hello World in JavaScript

console.log("Hello, World!");

Explanation: This code prints "Hello, World!" to the browser's console.

Chapter 3: JavaScript - DOM Manipulation

The Document Object Model (DOM) represents the structure of a web page. JavaScript can use the DOM to change the content, structure, and style of a document.

Example: Changing Text Content

// HTML: <p id="demo">Original Text</p>
document.getElementById("demo").textContent = "Updated Text";

Explanation: This code selects the paragraph element with the ID "demo" and changes its text content.

Chapter 3: JavaScript - Event Handling

JavaScript can respond to browser events such as clicks, mouse movements, or key presses. This makes web pages interactive.

Example: Button Click Event

// HTML: <button id="myButton">Click Me</button>
document.getElementById("myButton").addEventListener("click", function() {
  alert("Button was clicked!");
});

Explanation: This code adds an event listener to a button so that when it’s clicked, an alert pops up.

Chapter 3: JavaScript - Advanced Concepts

Modern JavaScript (ES6 and beyond) offers advanced features like arrow functions, Promises, async/await, and modules that help build robust applications.

Example: Arrow Functions & Promises

// Define a function that returns a Promise using an arrow function
const fetchData = () => {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      resolve("Data fetched successfully!");
    }, 2000);
  });
};

fetchData()
  .then(message => console.log(message))
  .catch(error => console.error(error));

Explanation: Here, an arrow function is used to define a function that returns a Promise. The Promise simulates an asynchronous operation (using setTimeout) and logs a message when resolved.

Chapter 4: Tools & Optimization - Build Tools

Build tools like Webpack, Babel, and NPM scripts help automate tasks such as bundling, transpiling, and minifying your code.

Example: Basic NPM Script

{
  "scripts": {
    "start": "webpack-dev-server --open",
    "build": "webpack --mode production"
  }
}

Explanation: This snippet from a package.json file defines NPM scripts to start a development server and build the project for production.

Chapter 4: Tools & Optimization - Performance Optimization

Performance optimization involves strategies such as lazy loading, code splitting, and image optimization to reduce load times and improve user experience.

Example: Lazy Loading Images

<img src="placeholder.jpg" data-src="actual-image.jpg" class="lazy">

<script>
  document.addEventListener("DOMContentLoaded", function() {
    const lazyImages = document.querySelectorAll("img.lazy");
    lazyImages.forEach(img => {
      img.src = img.getAttribute("data-src");
    });
  });
</script>

Explanation: In this example, a placeholder image is initially shown. Once the DOM is loaded, JavaScript replaces the placeholder with the actual image source.

Chapter 4: Tools & Optimization - Accessibility

Accessibility (a11y) ensures that your web content is usable by all people, including those with disabilities. This involves using semantic HTML, proper ARIA attributes, and ensuring keyboard navigation.

Example: Accessible Navigation

<nav aria-label="Main Navigation">
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#about">About</a></li>
  </ul>
</nav>

Explanation: This code provides navigation with an ARIA label to help assistive technologies understand the purpose of the section.
