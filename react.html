Chapter 1: Introduction to React

React is a JavaScript library for building user interfaces. It helps you create interactive UIs with reusable components.

Example: Rendering a Simple Element


// This example uses JSX to render a simple element
ReactDOM.render(
  <h1>Hello, React!</h1>,
  document.getElementById('interactive-output-1')
);

Explanation: This code renders a header with the text "Hello, React!" into the designated output container using ReactDOM.render.

Chapter 2: JSX - Introduction

JSX is a syntax extension for JavaScript that allows you to write HTML-like code in your JavaScript files. It makes your code easier to understand and maintain.

Example: Creating a JSX Element


// This example creates a JSX element and renders it.
const element = (
  <div>
    <h2>JSX Example</h2>
    <p>This is a JSX element.</p>
  </div>
);

ReactDOM.render(
  element,
  document.getElementById('interactive-output-2')
);

Explanation: JSX allows you to write HTML-like syntax directly in your JavaScript. The code is transpiled into standard JavaScript that creates React elements.
