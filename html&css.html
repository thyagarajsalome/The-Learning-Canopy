Chapter 1: HTML Basics - What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a webpage using elements and tags.

HTML Document Structure

A basic HTML document consists of the following elements: <!DOCTYPE html>, <html>, <head>, and <body>. These elements structure your document.

Example: Basic HTML Structure


<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First HTML Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a basic HTML document.</p>
  </body>
</html>

This example shows how to set up a simple HTML page with a title, heading, and paragraph.

Common HTML Tags

HTML provides a variety of tags to format and structure content. Some common tags include:

  • <h1> to <h6> for headings
  • <p> for paragraphs
  • <a> for hyperlinks
  • <img> for images
  • <ul>, <ol>, and <li> for lists

Example: Using Common Tags


<h1>Welcome to My Website</h1>
<p>This paragraph explains the content of the site.</p>
<a href="https://example.com">Visit Example.com</a>

HTML Attributes

Attributes provide additional information about HTML elements. They are added inside the opening tag.

Example: Using Attributes


<img src="image.jpg" alt="A descriptive text">
<a href="https://example.com" target="_blank">Open Example in a new tab</a>

HTML Forms and Inputs

Forms are used to collect user input. They include elements like <form>, <input>, <textarea>, and <button>.

Example: A Simple Form


<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <button type="submit">Submit</button>
</form>

Chapter 2: CSS Basics - What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows you to control the appearance of HTML elements.

Inline, Internal & External CSS

There are three ways to apply CSS to your HTML:

  • Inline CSS: Written directly in an HTML tag using the style attribute.
  • Internal CSS: Written inside a <style> tag within the HTML document.
  • External CSS: Written in a separate .css file that is linked to your HTML document.

Example: Inline and Internal CSS


<!-- Inline CSS Example -->
<p style="color: blue;">This text is blue.</p>

<!-- Internal CSS Example -->
<head>
  <style>
    p {
      font-size: 18px;
      color: green;
    }
  </style>
</head>

CSS Selectors and Properties

CSS selectors target HTML elements, and properties define the styles. For example, the color property sets the text color.

Example: Basic CSS Styling


body {
  background-color: #f4f4f4;
  color: #333;
  font-family: Arial, sans-serif;
}

h1 {
  color: #007BFF;
}

The CSS Box Model

The box model explains how an element's dimensions are determined by its content, padding, border, and margin.

Example: Box Model


div {
  width: 300px;
  padding: 20px;
  border: 5px solid #333;
  margin: 10px;
}

Colors, Fonts & Text Styles

CSS allows you to control text appearance by setting properties like font size, color, and family.

Example: Text Styling


p {
  font-size: 16px;
  color: #555;
  font-family: 'Helvetica Neue', sans-serif;
  line-height: 1.5;
}

Chapter 3: Advanced HTML & CSS - Semantic HTML & Accessibility

Semantic HTML uses meaningful tags (such as <header>, <nav>, <article>, and <footer>) to clearly define the different parts of a webpage. This improves accessibility and SEO.

Example: Semantic HTML Structure


<header>
  <h1>My Website</h1>
</header>
<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>
<article>
  <p>Main content goes here.</p>
</article>
<footer>
  <p>Copyright Â© 2025</p>
</footer>

Responsive Design & Media Queries

Responsive design ensures that your website adapts to different screen sizes. Media queries let you apply CSS styles based on device characteristics.

Example: Media Query


@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
  .sidebar {
    display: none;
  }
  .sidebar.active {
    display: block;
  }
}

Flexbox & CSS Grid

Flexbox and Grid are modern CSS layout modules that allow you to build responsive layouts with ease.

Example: Flexbox Layout


.container {
  display: flex;
  justify-content: space-between;
}
.item {
  flex: 1;
  margin: 10px;
}

Example: CSS Grid Layout


.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

Transitions & Animations

CSS transitions and animations let you animate changes in CSS properties to create engaging user experiences.

Example: CSS Transition


.button {
  background-color: #007BFF;
  transition: background-color 0.3s ease;
}
.button:hover {
  background-color: #0056b3;
}

CSS Custom Properties (Variables)

CSS variables allow you to store and reuse values throughout your stylesheet, making it easier to maintain consistent styling.

Example: Using CSS Variables


:root {
  --primary-color: #007BFF;
  --secondary-color: #6c757d;
}

body {
  background-color: var(--secondary-color);
  color: #fff;
}

a {
  color: var(--primary-color);
}

Chapter 4: JavaScript Integration - DOM Manipulation

JavaScript can manipulate the DOM to dynamically change HTML and CSS. This enables interactive web experiences.

Example: Changing Element Content


// HTML Element: <div id="demo">Original Content</div>
document.getElementById("demo").innerText = "Updated Content via JavaScript";

In this example, JavaScript selects an element with the id "demo" and updates its text content.

Interactive Examples with JavaScript

JavaScript can also add interactivity to your website. For example, you can toggle the sidebar on mobile devices.

Example: Toggle Sidebar


// Toggle sidebar visibility on mobile devices
const toggleBtn = document.getElementById("toggle-btn");
const sidebar = document.getElementById("sidebar");

toggleBtn.addEventListener("click", function() {
  sidebar.classList.toggle("active");
});

When the "Menu" button is clicked, the sidebar's "active" class is toggled to show or hide it.
